## Problem Statement

Users currently cannot authenticate to the system. We need secure login/logout functionality supporting multiple OAuth2 providers (Google, GitHub) to enable personalized features and data persistence.

## Success Criteria

- [ ] Users can sign in via Google OAuth2
- [ ] Users can sign in via GitHub OAuth2
- [ ] Session persists across browser refreshes
- [ ] Users can sign out and session is invalidated
- [ ] Protected routes redirect to login when unauthenticated
