CLI Reference

Shep is controlled entirely through its command-line interface. This reference covers all available commands.

Global Options​

OptionDescription
--help, -hShow help for a command
--version, -vPrint version number
--verboseEnable verbose output

Commands​

CommandDescription
featCreate and manage features
repoRepository management
agentManage and run agents
runRun an agent or task directly
sessionStart or manage interactive sessions
ideLaunch IDE integration
toolsList and configure agent tools
installInteractive setup wizard
uiLaunch the web UI
startStart background services
stopStop background services
restartRestart background services
statusShow service status
settingsView and edit configuration
versionPrint version information
upgradeUpgrade Shep to the latest version
