# Ship features while you sleep

Shep runs AI coding agents in parallel — each in its own git worktree — so you can develop 10+ features simultaneously. One command creates a branch, runs an agent, commits, pushes, and opens a PR.

## Quick start

```bash
npx @shepai/cli        # install and run
shep                   # launches dashboard at localhost:4050
shep feat new "add a /health endpoint" --push --pr
```

## How it works

You describe a feature in plain English. Shep creates an isolated worktree, assigns an AI agent, and manages the entire lifecycle — coding, committing, pushing, and opening a pull request. Run as many as you want at the same time.

Shep supports **Claude Code** (recommended), **Cursor CLI**, and **Gemini CLI** (coming soon).
