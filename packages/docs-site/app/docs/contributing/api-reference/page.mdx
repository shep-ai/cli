# API Reference

Technical specifications for Shep's core interfaces and domain types. All types are generated from TypeSpec definitions in `tsp/`.

- [Domain Models](/docs/contributing/api-reference/domain-models) -- Entities, value objects, and enums
- [Repository Interfaces](/docs/contributing/api-reference/repository-interfaces) -- Data access port definitions
- [Agent Interfaces](/docs/contributing/api-reference/agent-interfaces) -- Agent execution and orchestration interfaces
