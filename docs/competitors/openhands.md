## What It Does

OpenHands (formerly OpenDevin) is a composable platform for autonomous AI coding agents. It can take a GitHub issue and produce a working PR with minimal human guidance. Designed to scale from single tasks to thousands of parallel operations.

### Key Features

- **77.6% SWE-Bench Verified** — Strong benchmark performance
- **Multi-deployment** — Python SDK, CLI, web GUI, hosted cloud
- **Composable agents** — Define agent behavior in code
- **Model-agnostic** — Works with any LLM provider
- **Scales to 1000s** — Parallel operations for enterprise use

### What We Respect

OpenHands has built the largest open-source community around autonomous coding agents. Their composable architecture lets you build custom agent behaviors — it's infrastructure for the ecosystem. The MIT license and multi-deployment options lower the barrier for everyone.

### Where Shep Differs

OpenHands is a platform; Shep is a product. OpenHands gives you the building blocks to create your own agent workflows. Shep gives you a complete, opinionated lifecycle out of the box — install, run `shep feat new`, and you're going from idea to PR.
