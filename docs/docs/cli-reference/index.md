# CLI Reference

Shep is controlled entirely through its command-line interface. This reference covers all available commands.

## Global Options

| Option | Description |
|--------|-------------|
| `--help`, `-h` | Show help for a command |
| `--version`, `-v` | Print version number |
| `--verbose` | Enable verbose output |

## Commands

| Command | Description |
|---------|-------------|
| [`feat`](/cli-reference/feat) | Create and manage features |
| [`repo`](/cli-reference/repo) | Repository management |
| [`agent`](/cli-reference/agent) | Manage and run agents |
| [`run`](/cli-reference/run) | Run an agent or task directly |
| [`session`](/cli-reference/session) | Start or manage interactive sessions |
| [`ide`](/cli-reference/ide) | Launch IDE integration |
| [`tools`](/cli-reference/tools) | List and configure agent tools |
| [`install`](/cli-reference/install) | Interactive setup wizard |
| [`ui`](/cli-reference/ui) | Launch the web UI |
| [`start`](/cli-reference/start) | Start background services |
| [`stop`](/cli-reference/stop) | Stop background services |
| [`restart`](/cli-reference/restart) | Restart background services |
| [`status`](/cli-reference/status) | Show service status |
| [`settings`](/cli-reference/settings) | View and edit configuration |
| [`version`](/cli-reference/version) | Print version information |
| [`upgrade`](/cli-reference/upgrade) | Upgrade Shep to the latest version |
