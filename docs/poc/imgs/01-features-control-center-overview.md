# Features Control Center - Overview with Properties Panel

![Screenshot](01-features-control-center-overview.png)

## Summary

The main **Features Control Center** dashboard showing a canvas of feature cards in various lifecycle stages. The right-side **Properties** panel is open for the "Data Export Feature" card.

## Key Elements

- **Canvas layout**: Feature cards arranged in a free-form spatial layout, each showing its lifecycle stage (all currently in "Requirements"), a short description, a unique ID, and a progress percentage
- **Feature cards visible**: SSO Integration, Audit Logging, Dark Mode Support, Real-time Notifications, API Rate Limiting, 2FA Authentication, Data Export Feature, SSO Integration (duplicate)
- **Properties panel** (right side): Shows Overview/Activity/Settings tabs for the selected "Data Export Feature"
  - Feature name and description fields
  - **Attachments** section with 3 files: `requirements.pdf` (1.2 MB), `mockup.png` (845 KB), `api-spec.yaml` (12 KB)
  - **Team & Metadata** section (collapsed)
  - **Dependencies** section (collapsed)
- **"+ New Feature"** button at the bottom-left for adding features
