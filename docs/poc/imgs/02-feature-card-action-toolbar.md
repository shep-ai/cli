# Feature Card - Action Toolbar

![Screenshot](02-feature-card-action-toolbar.png)

## Summary

Close-up of a single feature card ("Dark Mode Support") with its **action toolbar** expanded at the bottom.

## Key Elements

- **Feature card**: Shows "Requirements" stage badge, title "Dark Mode Support", truncated description, ID `#f1i`, and 3% progress bar
- **Green "+" button**: Top-right corner for adding sub-items or related entities
- **"Start Dev Server" button**: Prominent action to launch the development environment
- **Toolbar icons** (bottom row, left to right): GitHub, grid/kanban view, globe/web preview, terminal, code editor, and a highlighted search/magnifier button
- The toolbar provides quick-access actions directly from the feature card without opening the full properties panel
