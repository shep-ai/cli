=== shep feat ls --show-archived — CLI command demo ===

# 1. Show ls command help — --show-archived flag available

$ shep feat ls --help
Usage: shep feat ls [options]

List features

Options:
  -r, --repo <path>  Filter by repository path
  --include-deleted  Include soft-deleted features
  --show-archived    Include archived features
  -h, --help         display help for command


# 2. First archive a completed feature

$ shep feat archive fc09efe0 --force
✓ Feature archived
  Name: Agent-Based Local Dev Environment
