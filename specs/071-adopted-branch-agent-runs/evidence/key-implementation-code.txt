KEY DECISIONS:
- AgentRun.status = pending (not running)
- AgentRun.prompt = "(adopted from existing branch)" (placeholder)
- Lifecycle = Review (if has PR) or Maintain (if no PR) — unchanged
- Spec directory preserved if already exists (handles partial work)
- Persistence order: AgentRun first, then Feature (referential integrity)
- No agent validation during adoption (deferred to StartFeatureUseCase)
