# Typography

The Shep AI design system uses system font stacks for optimal performance and native feel across platforms.

## Font Families

- **Sans** (`--font-sans`): `ui-sans-serif, system-ui, sans-serif, 'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol', 'Noto Color Emoji'`
- **Mono** (`--font-mono`): `ui-monospace, SFMono-Regular, 'SF Mono', Menlo, Consolas, 'Liberation Mono', monospace`
