# Design Tokens

The Shep AI design system uses CSS custom properties (design tokens) defined via Tailwind CSS v4's `@theme` directive. These tokens provide consistent styling across all components.

## Colors

|-------|------------|-----------|-------|
| `--color-background` | `#ffffff` | `#0a0a0a` | Page background |
| `--color-foreground` | `#0a0a0a` | `#fafafa` | Primary text |

## Usage in Components

These tokens are used throughout the component library via Tailwind CSS classes:

```tsx
// Using color tokens
<div className="bg-background text-foreground">
  <button className="bg-primary text-primary-foreground">
    Primary Button
  </button>
</div>

// Using spacing tokens
<div className="p-md gap-sm">
  Content with medium padding and small gap
</div>

// Using border radius
<div className="rounded-lg">
  Rounded container
</div>
```

## Dark Mode

Dark mode is implemented using the `.dark` class on the `<html>` element. The tokens automatically update when dark mode is active.

```tsx
// Enable dark mode programmatically
document.documentElement.classList.add('dark');

// Or use the useTheme hook
const { setTheme } = useTheme();
setTheme('dark');
```
